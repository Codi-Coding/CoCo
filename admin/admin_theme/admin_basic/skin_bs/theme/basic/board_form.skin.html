{# anc_bo_cmtpoint}
[{board.bo_table}] 댓글 {_levelset.gnu_name} 설정

중요 : 영카트5를 이용하시고 포인트 사용 설정이 [사용]으로 체크되어 있을 경우, 상품구매를 위해 결제 수단으로 사용될 수 있는 포인트입니다.

Note: 게시물의 첫번째 댓글에 주는 {_levelset.gnu_name}입니다. 0으로 설정하시면 기능이 작동하지 않습니다.
Note: 게시물이 작성이 되면 자동으로 포인트 지뢰가 심어집니다. 지뢰 폭탄을 터트릴 경우 주어지는 포인트입니다.
Note: 댓글을 작성할 경우, 지정한 확률에 따라 자동으로 행운의 포인트를 지급합니다.
{ frm_submit }
{# anc_bo_adopt}
[{board.bo_table}] 채택 게시판 설정

알림 : 채택 게시판을 사용하실 경우, 사이트 성격에 맞게 조건을 설정하실 수 있습니다.

Note: {_levelset.gnu_name}는 원글 작성자가 자신의 포인트의 일부를 글작성 후, 채택 적용시 채택된 회원에게 후원하는 {_levelset.gnu_name}입니다.
글작성 시, 자신이 지정한 포인트만큼 설정한 포인트를 사용합니다.
Note: 환급 {_levelset.gnu_name}비율을 설정하시면, 해당 비율만큼 제외하고 채택자에게 채택 {_levelset.gnu_name}를 후원합니다.
나머지는 질문 작성자 본인에게 환급됩니다.(채택율을 높이기 위해 설정 필요)
{ frm_submit }
